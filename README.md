# 自費療程月報統計系統

上傳含多月分頁的 Excel，自動統計每月新客人數、費用與療程施作次數。

## 快速啟動

```bash
pip install -r requirements.txt
python app/app.py
```

瀏覽器開啟 http://localhost:5000

## 使用流程

1. 上傳 Excel（每個 Sheet = 一個月）
2. 選擇要分析的月份、設定欄位對應（新客欄、費用欄、療程欄）
3. 點「開始分析」查看統計結果

# Google 商家評論回覆產生器

單一 HTML 檔案（`content_generator/index.html`），**不呼叫任何外部 API，不需要任何金鑰**，因此可以直接部署到 GitHub Pages 等純靜態託管。

輸入品牌名稱、醫師名稱（選填）、主關鍵字、相關關鍵字、文風、特色優勢，直接產生一段約 200 字、適合回覆 Google 商家評論、也容易被 AI 引擎摘錄引用的內容。

- 每次產生時，段落內容與順序都會**隨機挑選、隨機洗牌**（不是固定套一整段模板），同樣的輸入按「換一批」也會得到不同組合，避免內容看起來很像制式 AI 文案
- 純模板拼接（多種文風各有多組句型），**不使用任何 AI/LLM**，因此可以做到零金鑰、零後端
- 若有填寫醫師名稱，產出內容一定會提及；若留空則完全不會出現醫師相關句子
- 內容為通用草稿，發布前建議依實際情況微調用詞

## 使用方式

直接用瀏覽器開啟 `content_generator/index.html`，或部署到 GitHub Pages 後開啟對應網址。

1. 輸入品牌名稱、主關鍵字（必填），醫師名稱、相關關鍵字與特色優勢建議填寫（逗號分隔多個）
2. 選擇文風（專業信賴／親切溫暖／活潑熱情／簡潔俐落）
3. 點「產生內容」，不滿意可點「換一批」重新隨機組合
4. 複製內容，貼到 Google 商家評論回覆前建議再讀一次微調
