# 自費療程月報統計系統

上傳含多月分頁的 Excel，自動統計每月新客人數、費用與療程施作次數。

## 快速啟動

```bash
pip install -r requirements.txt
python app/app.py
```

瀏覽器開啟 http://localhost:5000

## 使用流程

1. 上傳 Excel（每個 Sheet = 一個月）
2. 選擇要分析的月份、設定欄位對應（新客欄、費用欄、療程欄）
3. 點「開始分析」查看統計結果

# AI 引用內容產生器（GEO Content Generator）

依據「awoo GEO 分析洞察」提出的四項提升 AI 引用機率規則，輸入關鍵字即可自動組裝出：

1. 文章頂部（時效性 + 作者權威）
2. 開頭摘要（三句話定義 + 中英術語並列）
3. H2 標題（全部改成問句，含在地詞與比較型問句）
4. 段落內容（數字 + 結論句 + 來源引導句）

## 正確性保護機制

為避免 AI 產出未經查證的醫學數字，本工具採「已審核知識庫」架構：

- 所有醫學數據（發生率、恢復期等）都存放在結構化知識庫中，並附上來源
- 新增的數據預設為「未審核」，**只有標記為「已審核」的數據才會被用於生成**
- Claude 只能使用你提供的已審核數據來組裝文章，系統提示明確禁止其自行捏造或延伸醫學數字
- 產出內容一律標示為草稿，需經醫療專業人員審閱後才能發布

## 快速啟動

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=your-api-key-here   # 或複製 content_generator/.env.example 為 .env
python content_generator/app.py
```

瀏覽器開啟 http://localhost:5001

## 使用流程

1. 在左側建立或選擇關鍵字（例：痔瘡手術），設定科別、中英術語、在地詞、比較主題
2. 新增醫學數據並標註來源，經醫療專業人員確認後點「標記已審核」
3. 在右側輸入醫師姓名（選填）、最後更新日期，點「產生內容」
4. 複製產出的草稿，交由醫療專業人員審閱後再發布
