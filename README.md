# 自費療程月報統計系統

上傳含多月分頁的 Excel，自動統計每月新客人數、費用與療程施作次數。

## 快速啟動

```bash
pip install -r requirements.txt
python app/app.py
```

瀏覽器開啟 http://localhost:5000

## 使用流程

1. 上傳 Excel（每個 Sheet = 一個月）
2. 選擇要分析的月份、設定欄位對應（新客欄、費用欄、療程欄）
3. 點「開始分析」查看統計結果

# AI 引用內容產生器（GEO Content Generator）

依據「awoo GEO 分析洞察」提出的提升 AI 引用機率規則，把院內人員填寫的內容自動組裝成方便 AI 引擎引用的格式：

1. 開頭摘要（三句話定義 + 中英術語並列）
2. 多個 H2 問答段落（標題皆為問句，含台灣在地詞與比較型問句，每段包含數字 + 結論句 + 來源引導句）

## 純靜態、不需要 API 金鑰

這是單一 HTML 檔案（`content_generator/index.html`），**不呼叫任何外部 API，不需要任何金鑰**，所有資料只存在瀏覽器的 localStorage：

- 醫學數字、結論、來源都由院內人員自行填寫並標記「已審核」
- 只有「已審核」的段落才會出現在產出內容中，避免未查證的內容外流
- 工具只負責依 GEO 規則組裝格式（開頭摘要、H2 問句化、在地詞/比較型問句提醒、來源引導句），不會自行生成或延伸任何醫學數字
- 因為不呼叫外部服務，可以直接部署到 GitHub Pages 等純靜態託管

## 使用方式

直接用瀏覽器開啟 `content_generator/index.html`，或部署到 GitHub Pages 後開啟對應網址。

1. 在左側建立關鍵字（例：痔瘡手術），填寫科別、中英術語、開頭摘要
2. 新增段落：H2 標題（問句）、數據、結論句、來源，經醫療專業人員確認後點「標記已審核」
3. 在右側輸入醫師姓名（選填）、最後更新日期，點「產生內容」
4. 複製產出內容，交由醫療專業人員最終審閱後再發布
