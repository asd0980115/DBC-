# 自費療程月報統計系統

上傳含多月分頁的 Excel，自動統計每月新客人數、費用與療程施作次數。

## 快速啟動

```bash
pip install -r requirements.txt
python app/app.py
```

瀏覽器開啟 http://localhost:5000

## 使用流程

1. 上傳 Excel（每個 Sheet = 一個月）
2. 選擇要分析的月份、設定欄位對應（新客欄、費用欄、療程欄）
3. 點「開始分析」查看統計結果

# Google 商家評論回覆產生器

單一 HTML 檔案（`content_generator/index.html`），**不呼叫任何外部 API，不需要任何金鑰**，因此可以直接部署到 GitHub Pages 等純靜態託管。

輸入主關鍵字、相關關鍵字、品牌名稱、文風、特色優勢，直接產生一段約 200 字、適合回覆 Google 商家評論、也容易被 AI 引擎摘錄引用的內容。

- 純模板拼接（多種文風各有多組句型，隨機組合），**不使用任何 AI/LLM**，因此可以做到零金鑰、零後端
- 按「換一批」可用同樣的輸入重新排列組合，得到不同版本，避免每則回覆長得一模一樣
- 內容為通用草稿，發布前建議依實際情況微調用詞

## 使用方式

直接用瀏覽器開啟 `content_generator/index.html`，或部署到 GitHub Pages 後開啟對應網址。

1. 輸入主關鍵字、品牌名稱（必填），相關關鍵字與特色優勢建議填寫（逗號分隔多個）
2. 選擇文風（專業信賴／親切溫暖／活潑熱情／簡潔俐落）
3. 點「產生內容」，不滿意可點「換一批」重新組合
4. 複製內容，貼到 Google 商家評論回覆前建議再讀一次微調
