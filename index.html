串接 Google Sheet — 自動產生新舊客、來源、療程、營收完整報表

① 連結 Google Sheet
🔧 第一次使用請先設定（一次即可）：
  1. 開啟你的 Google Sheet
  2. 點上方選單 擴充功能 → Apps Script
  3. 刪除原本的程式碼，貼上 apps-script.js 的內容（已在 GitHub 上）
  4. 點 部署 → 新增部署作業，類型選「網頁應用程式」
  5. 執行身分：我　　存取權：所有人
  6. 複製產生的 網頁應用程式網址，貼到上面輸入框
